A full-stack shopping list application that allows users to create and manage store-specific shopping lists with full CRUD functionality.
Made as the final project for CIS 2454 Full Stack Web Deb course at Oakland Community College. 

Live demo: https://alpedron.infinityfreeapp.com/ShoppingList/

Tech Used:
Frontend
* React (Vite)
* JavaScript (ES6+)
* CSS (custom styling, mobile-first)
  
Backend
* PHP (REST API)
* MySQL (InfinityFree hosting)
  
Hosting
* InfinityFree (Frontend + Backend + Database)
